# Installing the BreadCrumb_FileBrowser


## To run the project 
### 1. In the Server directory, you can run:
```
> cd server
> npm install
> npm start

The Server Runs on http://localhost:8000
```

### 2. In the Client directory, you can run:
```
> cd client
> npm install
> npm start

The Client Runs on http://localhost:3000
```
